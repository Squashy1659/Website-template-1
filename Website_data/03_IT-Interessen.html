IT soll interessant sein?

Ja! IT kann in der Tat Spaß machen. Denkt einfach mal darüber nach, was alles unter die Informationstechnik fällt. WhatsApp, YouTube, Instagram, Computerspiele, die Cloud, Amazon und noch vieles mehr sind direkte Angebote der IT. Mit nur ein wenig Wissensaneignung kann man schon komplett andere Betriebsprogramme verwenden, eine eigene Cloud und Websites betreiben, Windows Benutzer konstant darauf hinweisen, dass man Linux verwendet und wie viel besser es denn sei, Minecraft Server hosten und noch vieles mehr.

Betriebsprogramm GNU/Linux

Mein Interesse an Computern und den dazugehörigen Themen startete tatsächlich erst vor ca. 3 Jahren. Damals spielte ich mit virtuellen Maschinen herum und stieß dabei auf Linux. Wie so viele startete ich mit Ubuntu, wechselte viel herum und landete am Ende bei Arch Linux. I use arch btw ;)

Virtuelle Maschinen

Wie bereits erwähnt, stieß ich nur dank virtueller Maschinen auf Linux. Doch was ist das eigentlich? Virtuelle Maschinen erlauben es, einen oder mehrere virtuelle PCs auf dem eigenen Computer zu simulieren. Dazu werden die eigenen Ressourcen aufgeteilt. Dazu gibt es verschiedene Programme. Mit Programmen wie KVM+VirtManager sind annähernd Bare-Metal Leistungen erreichbar. Somit könnte man theoretisch macOS mit einer RX 7900 XT betreiben...

Docker

Wo wir doch gerade schon bei Virtualisierung sind, lasst mich ein paar Worte über Docker verlieren. Mit Docker lassen sich Anwendungen oder Betriebsprogramme in Container packen. Dies versichert, dass diese Anwendung auf jedem PC funktionieren, da alle Abhängigkeiten sich innerhalb von diesem Container befinden. Dabei wird jedoch, anders als bei VMs, der Kernel geteilt, wodurch nur wenige Systemressourcen verwendet werden. Im Bild sind momentan von mir benutzte Docker Container zu sehen.

Nextcloud

Mit der Hilfe von Docker Containern ist es auch relativ einfach eine eigene Cloud zu erschaffen. Der beliebteste Anbieter dabei ist Nextcloud, ein Kostenlos Community Programm, welches nur durch Sponsoren gehalten wird. Da jegliche Daten auf dem eigenen Server gespeichert werden, ist die gegebene Privatsphäre größer als es andere Anbieter anbieten können. Bei größeren Datenmengen ist dies auch meist auch günstiger als bei gemieteten Cloudanbietern.

Plex Media Server

Natürlich beschränken sich Docker Container nicht nur auf das Hosten einer eigenen Cloud. Es lässt sich auch eine eigene Art Netflix erstellen. Mit dem Plex Media Server können Filme, Musik, Sendungen und noch vieles mehr kostenlos gestreamt werden, vorausgesetzt man besitzt diese Medien bereits.
